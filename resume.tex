\section*{Flagship Project}

\project{RedSentinel: AI-Assisted XSS Vulnerability Scanner}{Final-year Capstone, Team of 3 \textbar\ \href{https://github.com/seamoonpandey/Xbow}{Code} \textbar\ \href{https://github.com/seamoonpandey/Xbow-report}{Report} \textbar\ \href{https://drive.google.com/file/d/1HZmdURP3l8JNoTyL40me_O1-8kat4Q9d/view?usp=sharing}{Demo}}
\textit{Lead architect; owned the NestJS orchestration core and the end-to-end ML pipeline. Teammates built the fuzzer and obfuscation engines.}
\begin{itemize}
\item Architected a polyglot microservice system: a \textbf{TypeScript/NestJS} core orchestrating three \textbf{Python/FastAPI} AI services over a \textbf{Redis/BullMQ} job queue, with \textbf{PostgreSQL} persistence and a real-time \textbf{Next.js} dashboard (Socket.IO).
\item Fine-tuned a \textbf{DistilBERT} multi-task classifier to label an injection point's context and severity, replacing brittle signature matching, and reached near-perfect accuracy on an \textbf{8,869-sample} held-out split of a deterministic, well-structured label space.
\item Built the full ML pipeline on a \textbf{59,122-entry} curated XSS dataset: data curation, custom tokenizer training, multi-task training with checkpointing, evaluation, and an inference API with ONNX/TorchScript export and confidence calibration.
\item Co-developed an \textbf{XGBoost} payload ranker (35 engineered features) to prioritize payloads by predicted effectiveness, with a heuristic fallback when the trained artifact is unavailable.
\item Designed the NestJS core: scan lifecycle, crawler, BullMQ queue processor, audit logging, JWT/API-key auth, WebSocket streaming, and HTML/JSON/PDF report generation, backed by unit and e2e test suites.
\end{itemize}
